### Android log colors

### Show line numbers


<img src="https://cdn-images-1.medium.com/max/800/1*SPjHi3YfjJrfZ-K47ZKeBA.png" alt="line_number" />

- `File > Settings > Choose Editor > General > Appearance > Tick on Show line numbers`
